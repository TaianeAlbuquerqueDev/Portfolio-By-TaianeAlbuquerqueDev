# 🗂️ Portfólio — TaianeAlbuquerqueDev

Portfólio pessoal desenvolvido com **HTML, CSS e JavaScript puro**, utilizando **Bootstrap 5** como biblioteca de estilos e **Web Components nativos** para componentização sem frameworks.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- [Bootstrap 5.3](https://getbootstrap.com/) — via CDN
- Web Components (Custom Elements API)

---


---

## 🧩 Componentes

O projeto utiliza **Web Components nativos** (Custom Elements) para reutilização de código sem depender de frameworks como React, Vue ou Angular.

| Componente | Tag | Descrição |
|---|---|---|
| Navbar | `<nav-bar>` | Barra de navegação responsiva com links para GitHub, Email e LinkedIn |
| Card de Tecnologia | `<card-tech>` | Card com ícone e nome de cada tecnologia |
| Carrossel | `<carousel-portfolio>` | Carrossel de imagens dos projetos |

### Exemplo de uso do card:

```html
<card-tech
  nome="Java"
  icone="https://cdn.jsdelivr.net/gh/devicons/devicon/icons/java/java-original.svg">
</card-tech>
```

## ▶️ Como executar

Não é necessário instalar nada. Basta abrir o arquivo `pages/index.html` diretamente no navegador, ou usar a extensão **Live Server** no VSCode para melhor experiência:

1. Instale a extensão [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) no VSCode
2. Clique com o botão direito em `pages/index.html`
3. Selecione **"Open with Live Server"**
